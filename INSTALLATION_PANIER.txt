⚙️ CONFIGURATION OPTIONNELLE

Pour modifier les prix, frais de port, taxes, etc., éditez:

CartController.php :
- Fonction addToCart() pour valider les prix
- Fonction getTotalPrice() pour les taxes

cart.php :
- Ligne "Frais de port: Gratuit" pour modifier la logistique

cart.css :
- Couleurs (#d4a574 = couleur or principale)

💡 PROCHAINES ÉTAPES

Pour améliorer le panier :

1. Ajouter un système de code promo
2. Ajouter des options de livraison
3. Créer une page de checkout/paiement
4. Ajouter des variantes de produits (taille, saveur)
5. Enregistrer les commandes en base de données
6. Système de wishlist
7. Panier persistant (compte utilisateur)
8. Intégration paiement (Stripe, PayPal)

❓ PROBLÈMES COURANTS

Q: Le compteur du panier n'apparaît pas
R: Vérifiez que cart.js est chargé et que Font Awesome est inclus

Q: Les articles ne s'ajoutent pas
R: Vérifiez que CartController.php est accessible
   Vérifiez la console du navigateur (F12) pour les erreurs

Q: La session se vide après rechargement
R: Normal, c'est une session PHP qui expire après fermeture du navigateur
   Pour persister, utiliser une base de données
